# Diabetes Readmission Prediction (CS4074)

## Project Overview

This project builds a distributed PySpark pipeline to predict whether diabetic patients will be readmitted within 30 days.

The pipeline is designed to handle noisy medical data and simulate real-world data challenges.

---

## Technologies Used

* PySpark (Apache Spark)
* Spark MLlib
* Python
* (Optional mention) Hadoop concepts

---

## Dataset

* UCI Diabetes Dataset
* 101,766 patient records
* 50 features

---

## Pipeline Steps

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Cleaning
4. Feature Engineering
5. Model Training (3 models)
6. Evaluation
7. Scalability Testing

---

## Models Used

* Logistic Regression ⭐ (Best)
* Decision Tree
* Random Forest

---

## Results

* Best Model: Logistic Regression
* F1 Score: 0.8387
* Accuracy: 0.888

---
